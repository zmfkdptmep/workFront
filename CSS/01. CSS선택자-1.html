CSS 선택자1

1. 모든 선택자(*)

HTML 문서안의 모든 태그를 선택할때 전체 선택자를 사용한다.
(pre태그 : 작성한 그대로 화면에 보여줌) 
    * {
        스타일 속성:값;
        스타일 속성:값;
        ...
    }

2. 태그 선택자

태그명을 사용해 요소를 선택 
        태그명{
            스타일 속성:값;
            스타일 속성:값;
            ...
        }

3. 아이디 선택자(#id)

요소의 id 속성을 사용하여 요소를 선택 
        #아이디명{
            스타일 속성:값;
            스타일 속성:값;
            ...
        }

4. 클래스 선택자(.class)

HTML 문서내에 여러 개의 태그를 동일한 클래스명으로 지정하여 선택할때 사용한다. 
        .클래스명{
            스타일 속성:값;
            스타일 속성:값;
            ...
        }
