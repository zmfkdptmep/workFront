8. text-shadow

텍스트에 그림자 효과를 추가
text-shadow : 가로 거리, 세로 거리, 번짐 정도, 색상

양수 : 글자의 오른쪽, 아래쪽
음수 : 글자의 왼쪽, 위쪽

번짐 정도 : 양수 값, 그림자가 모든 방향으로 퍼져 나가므로 크게 표현
HTML5 CSS JAVA SCRIPT
